# 모두의 학생부종합전형 시뮬레이터

정량(성적) + 정성(과세특) 데이터를 기반으로, 학생·학부모가 **입학사정관 관점의 열람/해석**을 연습할 수 있도록 만든 **교육용 가상 시뮬레이션**입니다.

## 실행 방법 (GitHub Pages)
1. 이 폴더의 파일을 그대로 GitHub 저장소에 업로드합니다.
2. `Settings → Pages`에서 `Deploy from a branch`를 선택합니다.
3. 브랜치를 `main` / 폴더를 `/ (root)`로 설정합니다.
4. 배포된 주소로 접속하면 바로 실행됩니다.

## 구성
- `index.html` : 메인 화면
- `assets/style.css` : 스타일(반응형, 고급 UI)
- `assets/app.js` : 동작 로직(학기 선택, 과세특 보기, 키워드 검색, 성적 추이)
- `data/applicantA.json` : 지원자 A 가상 데이터

## 안내
- 본 자료는 **교육용 가상 데이터**이며 실제 대학의 평가 시스템/합격 결과와 무관합니다.
